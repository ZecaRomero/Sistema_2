# Sistema de Gestão de Custo de Bovinos

Este é um sistema local (HTML/CSS/JavaScript) para controle de custo individual de bovinos.
Funciona sem banco de dados usando o `localStorage` do navegador.

## Como usar

1. Extraia o .zip
2. Clique duas vezes no arquivo `index.html` para abrir no navegador
3. Cadastre, edite ou exclua animais
4. Os dados ficam salvos no navegador automaticamente

## Estrutura

- `index.html` - Interface do sistema
- `style.css` - Estilo moderno e responsivo
- `script.js` - Lógica da aplicação
- `assets/fotos/` - Pasta para imagens dos animais
